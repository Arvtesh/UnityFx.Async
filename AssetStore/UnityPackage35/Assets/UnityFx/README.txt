SUMMARY
The package provides Unity3d extensions for asynchronous operations and coroutines
that can be used very much like Task Parallel Library (TPL).

LICENSE
https://opensource.org/licenses/MIT

INSTALLATION
No special actions needed: just extract the package content anywhere in your Assets folder.
