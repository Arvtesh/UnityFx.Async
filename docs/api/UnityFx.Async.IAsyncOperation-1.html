Result

Returns the result value of this operation. Accessing the property blocks the calling thread until the operation is complete. Read only.

Declaration
T Result { get; }
Property Value
Type Description
T
Remarks

Once the result of an operation is available, it is stored and is returned immediately on subsequent calls to the Result property. Note that, if an exception occurred during the operation, or if the operation has been cancelled, the Result property does not return a value. Instead, attempting to access the property value throws an System.InvalidOperationException exception.

Exceptions

System.InvalidOperationException

Thrown if the property is accessed before operation is completed.
