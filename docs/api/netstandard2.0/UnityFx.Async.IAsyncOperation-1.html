Result

Gets the operation result value.

Declaration
T Result { get; }
Property Value
Type Description
T

Result of the operation.
Remarks

Once the result of an operation is available, it is stored and is returned immediately on subsequent calls to the Result property. Note that, if an exception occurred during the operation, or if the operation has been cancelled, the Result property does not return a value. Instead, attempting to access the property value throws an InvalidOperationException exception.

Exceptions
Type Condition
InvalidOperationException

Thrown either if the property is accessed before operation is completed or if the operation has faulted.
