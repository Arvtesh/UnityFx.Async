ContinueWith<T>(T, Action<T, IAsyncCompletionSource>)

Creates a continuation that executes when the target IAsyncOperation completes.

Declaration
public static IAsyncOperation ContinueWith<T>(this T op, Action<T, IAsyncCompletionSource> action)where T : IAsyncOperation
Parameters
Type Name Description
T op

The operation to continue.
System.Action<T, IAsyncCompletionSource> action

An action to run when the op completes.
Returns
Type Description
IAsyncOperation

An operation that is executed after op completes.
Type Parameters
Name Description
T

Type of the operation to continue.
Remarks

The action is expected to start another asynchronous operation. When that operation is completed it should use the second action argument to complete the continuation. If the op is already completed the action is being called synchronously.

Continuation behaviour is very close to TPL: if SynchronizationContext is set the continuation posted onto it. Otherwise it is executed on a thread that initiated the operation completion.

Type Condition
ArgumentNullException

Thrown if the action is null.
