Parameters

An action to run when the op completes.
Returns
Type Description
IAsyncOperation

An operation that is executed after op completes.
Type Parameters
Name Description
T

Type of the operation to continue.
Remarks

The action is expected to start another asynchronous operation. When that operation is completed it should use the second action argument to complete the continuation. If the op is already completed the action is being called synchronously.

Continuation behaviour is very close to TPL: if SynchronizationContext is set the continuation posted onto it. Otherwise it is executed on a thread that initiated the operation completion.

Wait(IAsyncOperation, TimeSpan)

Waits for the IAsyncOperation to complete execution within a specified time interval.

Declaration
public static bool Wait(this IAsyncOperation op, TimeSpan timeout)
Parameters
Type Name Description
IAsyncOperation op

The operation to wait for.
TimeSpan timeout

A TimeSpan that represents the number of milliseconds to wait, or a TimeSpan that represents -1 milliseconds to wait indefinitely.
Returns
Type Description
Boolean

true if the operation completed execution within the allotted time; otherwise, false.
Exceptions
Type Condition
ArgumentOutOfRangeException

timeout is a negative number other than -1 milliseconds, or timeout is greater than MaxValue.
AggregateException

Thrown if the operation was canceled or faulted.
ObjectDisposedException

Thrown is the operation is disposed.
