TryAddCompletionCallback(AsyncOperationCallback, SynchronizationContext)

Attempts to add a completion callback to be executed after the operation has finished. If the operation is already completed the method does nothing and just returns false.

Declaration
bool TryAddCompletionCallback(AsyncOperationCallback action, SynchronizationContext syncContext)
Parameters
Type Name Description
AsyncOperationCallback action

The callback to be executed when the operation has completed.
SynchronizationContext syncContext

If not null method attempts to marshal the continuation to the synchronization context. Otherwise the callback is invoked on a thread that initiated the operation completion.
Returns
Type Description
Boolean

Returns true if the callback was added; false otherwise (the operation is completed).
Exceptions
Type Condition
ArgumentNullException

Thrown if the action is null.
ObjectDisposedException

Thrown is the operation has been disposed.

Completed

Raised when the operation has completed.

Declaration
event EventHandler Completed
Event Type
Type Description
EventHandler
Remarks

The event handler is invoked on a thread that initiated the operation completion (not on a thread that registered it). If the operation is already completed the event handler is not called.
