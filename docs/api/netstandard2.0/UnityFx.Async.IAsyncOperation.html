Interface IAsyncOperation

A disposable IAsyncResult with status information.

Namespace: UnityFx.Async

Syntax
public interface IAsyncOperation : IAsyncOperationEvents, IAsyncResult, IDisposable
Remarks

The interface is designed to be as close to the TLP Task as possible. All interface methods are expected to be thread-safe. The only exception to this rule is System.IDisposable.Dispose().
